# my-claude-marketplace

## 소스 다운로드
```
git clone https://github.com/seungkyua/my-claude-marketplace.git
cd ..
```


## claude marketplace 등록
```
claude

/plugin marketplace add ./my-claude-marketplace
```

## marketplace 로 부터 plugin 설치
```
/plugin install simple-e2e@simple-e2e-plugins
```

## 1. CLAUDE.md 복사
```
/simple-e2e:init-claude-md
```

## 2. prd 생성
```
/simple-e2e:prd-creator 컨퍼런스 홈페이지를 생성해줘.
```

## 3. sdd 생성
```
/simple-e2e:system-desgin
```

## 4. plan 생성
```
/simple-e2e:plan-gaenerator
```

## 5. 개발
```
plan.md 를 기반으로 모두 자동으로 구현해줘.
```
